🏭 Bảng giá Basecost

Khai báo giá xưởng theo loại sản phẩm — app tự tính lời cho từng đơn.

💵 Phí sàn & thuế (tính theo % giá bán)
App tự trừ các khoản này khi tính lời. Nhập 0 nếu không áp dụng.
🏭 Basecost theo loại sản phẩm
Khai báo giá xưởng theo loại SP. App tự tính: Lời = Giá bán − Basecost − Phí − Thuế.
🔗 Tự lấy bảng giá từ Sheet xưởng
Xưởng có Sheet bảng giá (cột loại SP + cột giá). Dán link để app tự lấy về.
💰 Lấy DOANH THU + giá + shop từ Merchize
Merchize lưu: revenue (doanh thu Etsy), basecost (giá xưởng), tag (tên shop). App lấy tự động & chính xác cho từng đơn. Cần đã cấu hình API key Merchize.

💸 Chi phí vận hành

Quảng cáo, lương, chi phí khác — app trừ vào lời để ra lời ròng.

💱 Tỷ giá USD → VNĐ
App tự lấy tỷ giá. Bạn có thể nhập tay (vd theo Vietcombank) để chính xác hơn.
📣 Quảng cáo theo ngày ($)
Ghi chi phí quảng cáo mỗi ngày. Quên ngày nào ghi bổ sung được (chỉ thêm, không sửa). Seller & admin đều ghi được.
🏢 Chi phí cố định (theo tháng, VNĐ)
Lương nhân viên, tiếp khách, thuê văn phòng... Nhập từng khoản riêng (VNĐ).

⚙️ Cài đặt — Kết nối Merchize
🏭 Đơn từ Merchize
Vào Merchize → menu API lấy Base URL + API Key. Dùng API Key (tab "API KEY") vì KHÔNG hết hạn — khỏi cập nhật hàng tháng.
📥 Lấy đơn cũ (mấy ngày trước)
Đơn cũ đã có trước khi cài webhook. Lấy về 1 lần:
1. Vào Merchize → trang Orders → bấm Export tải file CSV về máy
2. Tải file đó lên đây (cần đã điền API Key ở trên)
3. App tự đọc mã đơn + gọi Merchize lấy chi tiết đầy đủ
📦 Lấy Tracking từ Google Sheet xưởng
Xưởng có Google Sheet ghi mã đơn + tracking. App dò theo mã đơn, tự lấy tracking về.
1. Mở Sheet của xưởng → File → Share → Publish to web → chọn CSV → copy link
2. Dán link vào đây → Lưu → bấm "Đồng bộ tracking"
Sheet cần có cột chứa mã đơn (Order ID) và cột chứa tracking.
🛍️ Import đơn từ Etsy (CSV)
Lấy đơn trực tiếp từ Etsy — nguồn gốc, đầy đủ mọi đơn:
1. Etsy → Shop Manager → Settings → Options → Download Data → chọn Orders CSV → tải về
2. Chọn file CSV đó ở đây → bấm Import
Đọc: mã đơn, khách, địa chỉ, SKU, số lượng, tiền, trạng thái. (Tên SP/ảnh bổ sung sau từ Merchize.)
🔔 Webhook — đơn tự động về (khuyên dùng)
Cách tốt nhất: Merchize tự đẩy đơn mới sang app, không cần bấm gì. Làm 1 lần:
1. Copy URL webhook bên dưới
2. Vào Merchize → Setting → Webhook → Add Webhook → dán URL → chọn các sự kiện đơn hàng
3. (Tuỳ chọn) đặt Webhook Secret để bảo mật
💡 Để bảo mật webhook: đặt biến MERCHIZE_WEBHOOK_SECRET trên Vercel = chuỗi bí mật bất kỳ, rồi điền đúng chuỗi đó vào ô Secret khi tạo webhook ở Merchize.
